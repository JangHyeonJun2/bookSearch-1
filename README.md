# 개발해야하는 기능들

* 책 정보를 크롤링하여 mongoDB에 저장
* 회원가입시 mongoDB로 데이터 삽입
* 로그인시 데이터 호출로 회원 상태 확인
* 드롭다운 기능으로 인한 책 기능 선택, 선택시 결과값을 카드로 포스팅
